Frontend:React

HOW TO SETUP THE PROJECT?
1. Clone repository on your device
2. Download all depedencies by writing in new terminal
  ```js
  npm install
  ```
3. Build the docker files
```js
docker-compose up -d --build
```
4. Run the docker containers for backend and DB
